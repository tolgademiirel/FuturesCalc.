# FuturesCalc.

FuturesCalc., kripto vadeli işlemler için hazırlanmış tek dosyalı statik bir web uygulamasıdır.

## Özellikler
- Likidasyon hesaplama
- Kâr / zarar analizi
- Risk yönetimi odaklı giriş alanları
- Karanlık / açık tema desteği
- Mobil uyumlu arayüz

## Proje Yapısı
- `index.html` — uygulamanın tamamını içeren ana dosya

## Kullanım
Depoyu klonlayın veya dosyaları indirin, ardından `index.html` dosyasını tarayıcıda açın.

## Not
Bu proje istemci tarafında çalışan statik bir HTML/CSS/JavaScript yapısı olarak tasarlanmıştır.
